# Trades
Web application based on React and Flusk with SQLite DB
#### Init database from CSV
##### You should have:
- pandas
- sqlalchemy
- pathlib
- flask
- flask_cors
- flask_sqlalchemy

 1. Place your trades.csv file inside ./server/ folder
 2. Go to ./server folder
 3. Run import file
    ```
    python import.py 
    ```

#### Run server
1. Go to ./server folder
2. Run app file
    ```
    python app.py 
    ```

#### Run client app
1. Go to ./client
2. Install dependencies
    ```
    npm i 
    ```
3. Run app
    ```
    npm start
    ```
